##### Start MariaDB service: 
	`sudo systemctl start mariadb`

##### Change Mariadb password: 
	`mysqladmin password <newpassword>`

##### Login to MariaDB: 
	`mysql -u root -p'

##### Common Commands: 
	`show databases;`
	`show tables;`

##### Backup Mysql database: 
	'mysqldump database <name> > databasebackup.sql'
